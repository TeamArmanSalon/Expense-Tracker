# Expense Tracker

Expense Tracker is a simple, text-based financial management tool designed to help users record, organize, and analyze their personal finances. It provides an intuitive menu system for adding transactions, managing budgets, and generating financial summaries.

---

## Features

- **Add Expenses**  
  Record new expenses with a description, category, and amount.

- **Track Income**  
  Log income entries with source and amount.

- **Budget Management**  
  Set and manage budgets for different categories.

- **Transaction Records**  
  View all transactions, including both income and expenses.

- **Financial Summaries**  
  Display account balance and overall financial status.

- **Reports and Analysis**  
  Generate basic financial analysis and reports.

- **Menu System**  
  Navigate easily with a simple, text-based interface.

- **Settings and Controls**  
  Edit, delete, and manage records efficiently.

---

## Purpose

The Expense Tracker helps individuals maintain control over their finances by offering a straightforward way to monitor spending, track income, and stay within budget. It is ideal for anyone seeking a lightweight, no-frills solution to personal financial management.
